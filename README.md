# kotlin-cli-gradle-spring-findbugs-jacoco-pmd-mockito-junit-car-data

## Description
A POC for spring app using junit, findbugs,
jacoco, pmd, mockito framework.

## Tech stack
- kotlin
- gradle
  - spring
  - junit  
  - mockito
  - jacoco
  - pmd
  - findbugs

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
